# Generador de Trigramas en Python

Este proyecto implementa un generador de **trigramas** (secuencias de tres caracteres consecutivos) a partir de un texto. Es útil para tareas como búsquedas aproximadas, coincidencias difusas (fuzzy matching) o procesamiento de texto.

Incluye un ejemplo básico por consola y un ejemplo adicional con integración a una base de datos SQLite.
